## NVIDIA Isaac GR00T N1.6

<br>

![Model Architecture](/assets/img/2026-02-09/model-architecture.png)

<br>

NVIDIA Isaac GR00T N1.6은 범용 휴머노이드 로봇 기술을 위한 오픈소스 VLA 모델이다. 이 cross-embodiment 모델은 언어(language)와 이미지(images)를 포함한 멀티모달(multimodal) 입력을 받아 다양한 환경에서 조작(manipulation) 작업을 수행한다.

GR00T N1.6은 양손(bimanual) 로봇, 반인간형(semi-humanoid) 로봇, 그리고 광범위한 인간형(humanoid) 로봇 데이터셋을 포함한 다양한 로봇 데이터셋으로 학습되었다. 사후 학습(post-training)을 통해 특정 로봇 형태(embodiments), 작업(tasks) 및 환경(environments)에 맞게 조정할 수 있다.

GR00T N1.6의 신경망 구조는 시각-언어 파운데이션 모델(vision-language foundation model)과 연속적인 동작의 노이즈를 제거하는 확산 트랜스포머 헤드(diffusion transformer head)를 결합한 구조이다.

<br>

## Clone the Repository

<br>

`$ git clone https://github.com/NVIDIA/Isaac-GR00T.git`

`$ cd Isaac-GR00T`

<br>

## Set Up the Environment

<br>

`$ conda create -n gr00t python=3.10`

`$ conda activate gr00t`

`$ pip install -e .`

`$ pip3 install torch==2.7.0 torchvision==0.22.0 torchaudio==2.7.0 --index-url https://download.pytorch.org/whl/cu128`

`$ pip install --no-build-isolation flash-attn==2.8.0.post2`

`$ pip install decord`

<br>

※ flash_attn: 2.8.0.post2 / torch: 2.7.0+cu128 / transformers: 4.51.3 (RTX 5080)
